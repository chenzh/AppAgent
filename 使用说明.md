### 切换到完整版：
```bash

# 停止当前应用（Ctrl+C）
# 运行完整版
python3 app.py
```

### 切换到简化版：
```bash
# 安装LibreOffice

# 停止当前应用（Ctrl+C）
# 运行简化版
python3 app_simple.py
```
